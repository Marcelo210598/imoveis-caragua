All commands in this workflow are safe to auto-run.

2. Generate store assets (icon 512x512, feature graphic 1024x500).
3. Update the Play Store checklist in `historico/mobile-app/checklist-play-store.md`.
4. Update `progress.md` with latest sprint info.
